Getting Started

To use oform on web page just add script tag

<script src= 'path to library /oform.js' ></script>

Create an empty element to place content that will be generated. Set id attribute that will be used while calling the library.

< div id= 'this id will be used by library' > < /div >

Now it's possible to use oform.js functions in custom scripts.
