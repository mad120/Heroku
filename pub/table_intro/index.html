Tables

Table generator accepts array of objects and automatically adapat to object type.

Each table row has 'click' event that stores row data to the buffer adn could be accessed by : oform.selected()

To clear the slected buffer : oform.clear()

Examples for table generator usage :

oform.append(id).table(data).sortable().red();

oform.append(id).table(data).filtered().avg();

To generate table with another data set select situable row in table.
